서버의 규칙

다른 사람의 물건을 허락 없이 가져가지 말아주세요. (허락 받은 경우는 제외) 특정 유저를 괴롭히지 말아주세요. (ex. 특정 유저를 반복적으로 킬하기) 욕이나 비속어를 삼가주세요. (초성 포함) 핵이나 엑스레이를 사용하지 말아주세요. (관리자 허락 시 가능)

서버의 특징

컴퓨터 & 모바일 통합 기기에 상관없이 모바일 환경에서도 편하게 사용 가능하도록 통합되었습니다. 빠른 대처와 대응 문제가 생기면 관리자에게 문자로 부탁드립니다. (신고 포함)

서버 입장 정지 (3시간)

관리자에게 3번 이상 적발된 경우 핵이나 엑스레이를 사용한 경우 서버의 취약점을 악용한 경우 * 정지 시 관리자에게 문자로 "항소"를 보낼 수 있습니다.

서버 접속법

[모바일] 1. 'BedrockTogether' 앱 설치 및 실행 2. 첫 번째 칸: be.neocraft.p-e.kr 입력 3. 두 번째 칸: 19132 입력 4. 광고 시청 후 마인크래프트 실행 -> 'LAN 서버' 접속 [자바/컴퓨터] 1. 마인크래프트 최신 릴리즈 실행 2. 멀티플레이 -> 서버 추가 3. 주소창: neocraft.p-e.kr (안될 시 sub.neocraft.p-e.kr 나 play.neocraft.p-e.kr)
